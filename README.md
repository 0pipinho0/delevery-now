# Назначение продукта 
Приложение преднозначено для доставки еды, вещей, одежды, автомобилей со всей территории РФ и не только.
# Использование
- Нажатие на кнопку Войти пребрасывает вас на этап введения логина и пароля акаунта
- Нажатие на кнопку Регистрация пребрасывает вас на этап регистрации пользователя
- Нажатие на раздел Доставка показывает вам каталог товаров
- Нажатие на раздел заказы показвает вам ваши заказы
- Нажатие на кнопку Предыдущие заказы показывает вам ваши предыдущие заказы
# Технические требования
- Android не меньше 10 версии
- 2 гб оперативной памяти
- 256мб памяти на устройстве
- Стабильное интернет подключение
# Дальнейшее развитие
В будущем мы собираемся изменить дизай под ваши предпочтения и организовать доставку из ещё больших стран.
